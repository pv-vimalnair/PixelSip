# Use Cases

## Primary users

PixelSip is intended for people who spend long periods working or studying in Chrome and want a lightweight reminder without maintaining a health-tracking application.

## Use case 1: Focused desk work

**Context:** A user spends several hours working in browser-based tools.

**Flow:**

1. PixelSip's pinned toolbar glass begins full.
2. The user works while the glass gradually drains.
3. At one hour, the icon shakes and a notification appears.
4. The user drinks water and clicks **I drank water**.
5. The glass refills and the next hour begins.

**Value:** The reminder stays visible without interrupting focus before it is needed.

## Use case 2: Reminder arrives at a bad moment

**Context:** The user is in a meeting when the glass becomes empty.

**Flow:**

1. PixelSip displays the reminder.
2. The user selects **Remind me in 10 min**.
3. A short timer runs.
4. PixelSip reminds the user again and waits for confirmation.

**Value:** The user can defer the reminder without losing it.

## Use case 3: Sleeping or offline hours

**Context:** The user does not want reminders overnight.

**Flow:**

1. Quiet hours begin while 23 minutes remain.
2. PixelSip freezes the timer at 23 minutes.
3. Quiet hours end in the morning.
4. PixelSip resumes with the same 23 minutes remaining.

**Value:** The extension respects sleeping hours without discarding timer progress.

## Use case 4: Intentional pause

**Context:** The user temporarily wants no hydration reminders.

**Flow:**

1. The user opens the popup and selects **Pause**.
2. The toolbar icon becomes muted and the remaining time freezes.
3. The user selects **Resume** later.

**Value:** Reminder control remains explicit and reversible.

## Non-goals

- Measuring actual water intake
- Giving medical advice
- Calculating personalized hydration requirements
- Tracking streaks or daily totals
- Reading browsing activity or page content
- Requiring an account
