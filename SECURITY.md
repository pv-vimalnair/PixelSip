# Security Policy

## Reporting a vulnerability

Please report security or privacy issues privately by emailing `pv.vimalnair@gmail.com`.

Include:

- A clear description of the issue
- Steps to reproduce it
- The affected PixelSip version
- Any potential user impact

Please do not publicly disclose unresolved vulnerabilities before there has been a reasonable opportunity to investigate and address them.

## Security posture

PixelSip:

- Uses no remote code
- Requests no host permissions
- Has no backend or authentication system
- Stores reminder preferences locally
- Does not collect or transmit user data
